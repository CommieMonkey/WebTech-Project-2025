Webserver hosting

Hosting van een kleine website op Linux

Opzetten van een webserver met virtuele hosts en HTTPS voor een testsite.

Projectoverzicht

  • Installatie van Apache of Nginx op Linux
  • Virtuele hosts voor meerdere sites
  • Directorystructuur per website
  • Self-signed certificaat voor HTTPS in een labo

Tools & technologieën

  • Linux (bijv. AlmaLinux / Ubuntu)
  • Apache / Nginx
  • HTTP & HTTPS
  • OpenSSL (self-signed certs)

Aanpak

  1. Installeren en updaten van de webserver op een Linux-VM.
  2. Aanmaken van directories voor elke site onder /var/www.
  3. Configureren van virtuele hosts met aparte servernamen en documentroots.
  4. Genereren van een self-signed certificaat en config voor een HTTPS-vhost.
  5. Testen van de sites via een hosts-file en het analyseren van de logs.

Resultaat

De webserver kan meerdere sites tegelijk hosten, zowel via HTTP als HTTPS. Dankzij een duidelijke structuur is het eenvoudig om een extra site toe te voegen of om logs te analyseren.
