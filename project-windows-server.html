Windows Server-domein voor een kleine organisatie

Active Directory, DNS, shares en NTFS-rechten.

Samenvatting

  • Opzetten van een Active Directory-domein.
  • Structuur met OU’s per afdeling.
  • DNS geïntegreerd in AD.
  • Shares met NTFS- en share-permissies.

Gebruikte technologieën

  • Windows Server
  • Active Directory Domain Services
  • DNS, DHCP
  • NTFS-permissies

Aanpak

  1. Installatie van Windows Server en promotie tot domain controller.
  2. Aanmaken van OU’s per afdeling (IT, HR, Sales, …).
  3. Gebruikers en groepen aanmaken, koppelen aan de juiste OU.
  4. Inrichten van shares (bijv. \\server\afdeling) met NTFS- en share-rechten.
  5. Clients joinen aan het domein en testen van login, shares en groepslidmaatschap.

Resultaat

De organisatie beschikt over een centraal beheerd domein waarin gebruikers en computers consistent worden beheerd. Toegang tot netwerkshares is duidelijk afgebakend, en beheerderstaken zijn eenvoudiger door de OU-structuur en groepsrechten.
